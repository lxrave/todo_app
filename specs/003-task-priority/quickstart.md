# Quickstart: Приоритет задачи (003-task-priority)

**Ветка**: `003-task-priority`  
**Дата**: 2026-02-17

Краткие команды для разработки и проверки после реализации.

---

## Предусловия

- Реализована фича 001-todo-list (список задач, добавление, переключение выполнения, перетаскивание).
- Node.js 18+, npm или pnpm.

---

## Установка и запуск

```bash
npm install
npm run dev
```

Открыть в браузере URL (обычно `http://localhost:5173`).

---

## Проверка по спецификации

После реализации убедиться:

1. **Приоритет по клику (US1)**  
   У каждой задачи отображается приоритет (Low / Medium / High). Клик по значению открывает выпадающий список; выбор нового значения обновляет задачу и перемещает её в соответствующую группу (в конец). После перезагрузки приоритет сохраняется.

2. **Сортировка по приоритету (US2)**  
   Список отображается в порядке: сначала все High, затем Medium, затем Low. Карточки имеют еле уловимый цвет: High — красный, Medium — жёлтый, Low — синий. При смене приоритета задачи она оказывается в конце новой группы.

3. **Перетаскивание в рамках группы (US3)**  
   Перетаскивание меняет порядок только внутри одной группы приоритета. Попытка отпустить задачу в зоне другой группы не срабатывает (drop недоступен). Порядок внутри группы сохраняется после перезагрузки.

4. **Новая задача**  
   У новой задачи приоритет по умолчанию Medium; он отображается и сохраняется.

5. **Граничные случаи**  
   Закрытие выпадающего списка без выбора (клик вне, Escape) оставляет прежний приоритет. Одна задача в группе — перетаскивание не меняет порядок.
