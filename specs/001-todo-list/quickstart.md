# Quickstart: Todo List (001-todo-list)

**Ветка**: `001-todo-list`  
**Дата**: 2026-02-16

Краткие команды для разработки и проверки после реализации.

---

## Предусловия

- Node.js 18+ (рекомендуется LTS)
- npm или pnpm

---

## Установка зависимостей

```bash
npm install
# или
pnpm install
```

---

## Запуск в режиме разработки

```bash
npm run dev
# или
pnpm dev
```

Открыть в браузере URL, указанный в выводе (обычно `http://localhost:5173`).

---

## Сборка для продакшена

```bash
npm run build
# или
pnpm build
```

Артефакты — в каталоге `dist/`.

---

## Запуск тестов

```bash
npm run test
# или
pnpm test
```

Для watch-режима:

```bash
npm run test:watch
# или
pnpm test:watch
```

---

## Линтинг и форматирование

```bash
npm run lint
# или
pnpm lint
```

(Команды зависят от настроек проекта: ESLint, Prettier и т.д.)

---

## Проверка по спецификации

После реализации убедиться:

1. **Добавление задачи** (US1): ввод текста → «Добавить» → задача появляется в списке.
2. **Просмотр списка** (US2): пустой список и список с задачами отображаются корректно; виден текст и статус.
3. **Отметка выполнения** (US3): переключение статуса задачи; после перезагрузки страницы состояние сохраняется.
4. **Переупорядочивание** (US4): перетаскивание меняет порядок; после перезагрузки порядок сохранён.
5. **Пустой ввод** (FR-006): кнопка неактивна или показывается сообщение; пустая задача не создаётся.
6. **Данные** (FR-004): после перезапуска приложения список и порядок восстанавливаются из localStorage.
