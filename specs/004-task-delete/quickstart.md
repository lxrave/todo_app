# Quickstart: Удаление задачи (004-task-delete)

**Ветка**: `004-task-delete`  
**Дата**: 2026-02-16

Краткие команды для разработки и проверки после реализации.

---

## Предусловия

- Реализованы фичи 001-todo-list и 003-task-priority (список задач, приоритеты).
- Node.js 18+, npm.

---

## Установка и запуск

```bash
npm install
npm run dev
```

Открыть в браузере `http://localhost:5173`.

---

## Проверка по спецификации

После реализации убедиться:

1. **Иконка корзины**  
   На каждой карточке справа внутри карточки видна иконка корзины (всегда, в т.ч. на мобиле). Текст задачи не перекрывает иконку.

2. **Диалог подтверждения**  
   Клик по иконке открывает диалог с текстом «Удалить задачу «[название задачи]»?» и кнопками «Отмена» и «Удалить».

3. **Подтверждение удаления**  
   Нажатие «Удалить» закрывает диалог, задача исчезает из списка. После перезагрузки страницы эта задача отсутствует.

4. **Отмена**  
   Нажатие «Отмена», Escape или клик вне диалога закрывает диалог, задача остаётся в списке.

5. **Пустой список**  
   После удаления последней задачи отображается пустое состояние списка.
